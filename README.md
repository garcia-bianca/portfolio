# Portfólio Pessoal

Este é o meu portfólio pessoal desenvolvido para apresentar meus projetos, minhas habilidades em desenvolvimento frontend e meus links profissionais.  
O site reúne informações sobre mim, tecnologias que estudo e projetos práticos desenvolvidos com HTML, CSS e JavaScript.

## Sobre o projeto

O objetivo deste portfólio é centralizar minha apresentação profissional em um só lugar, facilitando o acesso aos meus projetos, repositórios e informações de contato.[web:1213][web:1214]

## Como executar o projeto

1. Clone este repositório:
```bash
git clone https://github.com/garcia-bianca/portfolio.git
```

2. Acesse a pasta do projeto:
```bash
cd portfolio
```

3. Abra o arquivo `index.html` no navegador
